# Templates

Dandori provides optional, lightweight templates to help teams pilot the framework without vendor or tool assumptions. The templates live in the repository under `templates/`.

Included templates
- Intent — `templates/intent-template.md`
- Specification — `templates/spec-template.md`
- Decision Request — `templates/decision-request-template.md`
- Integration Review — `templates/integration-review-template.md`
- Spec Retrospective — `templates/spec-retro-template.md`

Browse the templates in the repository folder or on GitHub:
- https://github.com/GTuritto/dandori/tree/main/templates

These scaffolds are optional. Adapt them to your context and keep them light. For contributions, see `CONTRIBUTING.md`.
