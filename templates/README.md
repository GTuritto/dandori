# Templates

Practical, lightweight, tool-agnostic scaffolds to help teams pilot Dandori. Use as-is or adapt to your context. These templates are optional and do not change the framework’s principles.

- Intent: intent-template.md
- Specification: spec-template.md
- Decision Request: decision-request-template.md
- Integration Review: integration-review-template.md
- Spec Retrospective: spec-retro-template.md

Contributions are welcome. See ../CONTRIBUTING.md.
